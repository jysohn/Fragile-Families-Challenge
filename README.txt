FragileFamilies.py and FragileFamilies2.py require the dataset from the fragiile families challenge: https://fragilefamilies.princeton.edu/

After running the preprocessing file provided by the course instructors, make sure the file "output.csv" is in the same directory as the file.

Similarly, in order to make sure the test data also imports correctly, be sure to include "test.csv", which should also have been cleaned by the preprocessing file, run correctly as well.

The comments outline the general theme of section, starting with imports, then baseline models, then feature selection methods, and finally, results and graphs.
